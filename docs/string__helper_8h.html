This function compares two wide strings ignoring non-alnum wide characters.

Parameters
[in]left_wstr,right_wstrThe wide strings this function will compare
Returns
< 0 if left_wstr < right_wstr, 0 if left_wstr == right_wstr, > 0 if left_wstr > right_wstr
Remarks
This function considers an empty wide string to be greater than any other wide string, and a prefix wide string to be greater than any wide string it is a prefix of
